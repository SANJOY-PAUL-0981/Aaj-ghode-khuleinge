What Are JavaScript Classes?

Let's understand this with a simple example first-


Imagine you have a toy factory. In this factory, you want to create lots of the same toy, like toy cars. You want all the toy cars to be the same in some ways, like they all have wheels, but they might be different colors. To make things easier, you create a blueprint for the toy car. This blueprint tells the factory how to make the toy car.


So, classes are a way to define the blueprint for creating objects(the toy). It tells the computer how to create objects (like toy cars) with the same basic parts, but each object can still be a little different.

Let's try to understand the terminologies:

class ToyCar: We're making a new class called ToyCar. This is our blueprint.

constructor(wheels,doors,colour): This is a special function that gets called when we make a new toy car. It sets up the car's basic features, like the number of wheels, doors and its color.

this.wheels = wheels: Every car we make with this blueprint will have wheels.

this.color = color: Each car can have a different color according to your input.

totalParts(): This is a function which is calculating total parts (wheels + doors) and returning the amount of parts needed from inside of the class

printColour(): This is a function which is printing the colour of the car from inside of the class
