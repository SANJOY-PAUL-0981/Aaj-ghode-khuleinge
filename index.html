JavaScript Promises

What Are JavaScript Promises?

Let's understand this with a simple example first-


A JavaScript Promise is like a promise you make in real life. Imagine you ask a friend for help, and they promise to help you later. You don't know when exactly they'll help, but you trust that they will. In JavaScript, a Promise is a way to handle things that take time, like downloading a file or getting data from the internet. When you start one of these tasks, you create a Promise.

Definition:

In JavaScript, a Promise is an object that represents the ultimate completion (or failure) of an asynchronous operation and the value it returns. Promises are used to handle asynchronous activities more effectively than standard callback functions, making it easier to manage code that runs asynchronously, such as API requests, file I/O, or timers.

What's Happening?

You're making a special function called 'setTimeoutPromisified'. It takes a number 'ms' (which stands for milliseconds) as input. 1000 milliseconds = 1 second, so 3000 milliseconds = 3 seconds. Inside this function, you create a 'Promise'. The Promise waits for the number of milliseconds you give it (in this case, 3000 milliseconds). After the time is up, it says, “Okay, I'm done waiting!” by calling 'resolve'.


This is another 'function' called 'callback'. When this 'function' is called, it will print "3 seconds have passed" on the screen.

You call the 'setTimeoutPromisified' function with 3000 milliseconds (3 seconds). The 'Promise' starts waiting for 3 seconds. After 3 seconds, the 'Promise' is fulfilled, and it calls the 'callback' function. The 'callback' function runs, and it prints "3 seconds have passed."

FAQ

  • What is this in JavaScript class?

    In JavaScript classes, this refers to the instance of the class that the method is being called on. It allows you to access and manipulate the properties and methods of that specific object.

  • What is constructor in JavaScript class?

    In JavaScript classes, a constructor is a special method used for creating and initializing objects created with that class.

  • What is 'new' in JavaScript class?

    In JavaScript the new is a keyword which used to create an instance of an object from a constructor function or a class. Basically it creates a new empty object.

  • What is instances in JavaScript class?

    In JavaScript, an instance is a unique copy of an object created from a class or constructor function. It represents a specific realization of that blueprint, with its own set of properties and values.

  • What is resolve in JavaScript promise?

    The resolve() function is used to signal that the asynchronous operation has completed successfully. It takes a single argument, which is the value that the Promise will be fulfilled with. This value can be anything - data, an object, another Promise, etc. Consuming a Promise.

  • What is .then in JavaScript promise?

    In JavaScript, .then() is a method available on Promise objects. It's used to handle the eventual result of an asynchronous operation, whether it's successful (resolved) or unsuccessful (rejected).
